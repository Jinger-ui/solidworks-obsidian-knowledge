## 详细笔记

> 以下内容基于西电版《数字信号处理》教材知识体系撰写，对应 B 站分 P 标题「6-2数字滤波器及原理」。**非 UP 逐字转写**；视频中的精选例题请对照观看。

### 1. 数字滤波器分类

| | IIR | FIR |
|--|-----|-----|
| 冲激响应 | 无限长 | 有限长 |
| 系统函数 | 有理 $H(z)$ | $H(z)$ 仅零点 |
| 稳定性 | 需设计保证 |  inherently 稳定 |
| 相位 | 非线性（一般） | 可线性相位 |
| 阶数 | 较低 | 较高 |

### 2. 设计流程

**指标** → **选型** → **模拟原型** → **变换** → **数字 $H(z)$** → **实现结构**

### 3. 性能指标

- **通带波纹** $A_p$（dB）
- **阻带衰减** $A_s$（dB）
- **截止频率** $\omega_p,\omega_s$
- **过渡带宽度**

### 4. IIR 设计方法

- 脉冲响应不变法
- 双线性变换法
- 频率变换（高通/带通）

### 5. FIR 设计方法

- 窗函数法
- 频率采样法
- 等波纹（Parks-McClellan，本课可能略）

### 6. 典型例题

**例**：要线性相位低通，选 IIR 还是 FIR？

**FIR**——IIR 一般非线性相位；FIR 可设计 Type I～IV 线性相位。

### 7. 考试要点

- 对比 IIR/FIR 优缺点
- 叙述完整设计流程
- 理解指标物理意义
- 为后续各设计方法铺垫

### 8. dB 指标换算

$A_p=-20\log_{10}(\delta_p)$（通带波纹），$A_s=-20\log_{10}(\delta_s)$（阻带衰减）。给定 dB 先还原 $\delta$ 再代入阶数公式。过渡带越窄、波纹越小，所需阶数越高——IIR/FIR 选型题常考此权衡。

### 9. 指标到实现的完整链路

模拟指标（dB）$\to$ 归一化 $\Omega$ $\to$ 模拟 $H_a(s)$ $\to$ 双线性/脉冲不变 $\to$ 数字 $H(z)$ $\to$ 级联二阶节 + 系数量化 $\to$ 仿真验证幅频/相频。任一步遗漏预畸变都会导致数字截止频率偏移。

### 9. 选型决策树

要线性相位 $\to$ FIR；要低阶陡峭 $\to$ IIR；要绝对稳定且允许长阶数 $\to$ FIR 窗函数；要实时低延时 $\to$ IIR 直接 II + 定点实现。
