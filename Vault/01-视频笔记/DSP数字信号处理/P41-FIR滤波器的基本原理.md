## 详细笔记

> 以下内容基于西电版《数字信号处理》教材知识体系撰写，对应 B 站分 P 标题「7-1FIR滤波器的基本原理」。**非 UP 逐字转写**；视频中的精选例题请对照观看。

### 1. FIR 系统函数

$$H(z)=\sum_{n=0}^{N-1}h(n)z^{-n}$$

**仅零点**（$z=0$ 处 $N-1$ 阶极点可抵消），**无反馈**，**永远 BIBO 稳定**。

### 2. 冲激响应

$h(n)$ 长度 $N$，$n\ge N$ 时 $h(n)=0$。

$$y(n)=\sum_{k=0}^{N-1}h(k)x(n-k)$$

有限卷积，易实现。

### 3. 线性相位条件

$$h(n)=\pm h(N-1-n)$$

**偶对称** → Type I/II 线性相位；**奇对称** → Type III/IV。

群延迟 $\tau=(N-1)/2$ 样本（常数）。

### 4. 线性相位意义

各频率分量延迟相同，**波形无畸变**（重要于音频、通信）。

### 5. 与 IIR 对比

FIR 相位可控、稳定，但同样过渡带需**更高阶** $N$。

### 6. 典型例题

**例**：$N=5$，$h=\{1,2,3,2,1\}$，是否线性相位？

$h(0)=h(4)=1$，$h(1)=h(3)=2$，$h(2)=3$ 对称 → **是**，Type I。

### 7. 考试要点

- 写 FIR $H(z)$ 一般形式
- 线性相位对称条件（必考）
- 四类线性相位 FIR 特点
- FIR 稳定性的根本原因
