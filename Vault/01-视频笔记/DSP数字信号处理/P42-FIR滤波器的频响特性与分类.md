## 详细笔记

> 以下内容基于西电版《数字信号处理》教材知识体系撰写，对应 B 站分 P 标题「7-2FIR滤波器的频响特性与分类」。**非 UP 逐字转写**；视频中的精选例题请对照观看。

### 1. FIR 频率响应

$$H(e^{j\omega})=\sum_{n=0}^{N-1}h(n)e^{-j\omega n}=e^{-j\omega(N-1)/2}\cdot H_r(\omega)$$

线性相位时 $H_r(\omega)$ 为实函数（幅度特性）。

### 2. 四类线性相位 FIR

| 类型 | 对称性 | $N$ | $h(0)$ 处 |
|------|--------|-----|----------|
| I | 偶 | 奇/偶 | 任意 |
| II | 偶 | 偶 | 任意 |
| III | 奇 | 偶 | 0 |
| IV | 奇 | 奇 | 0 |

### 3. 零点约束

Type III/IV 在 $\omega=0$ 或 $\omega=\pi$ 处有**强制零点**（如 Type III 在 $\omega=0$ 处 $H=0$，不能做高通）。

### 4. 选型指南

- **低通**：Type I 最灵活
- **高通**：Type I/II（$N$ 奇），避免 Type III
- **带通**：Type I/III
- **微分器**：Type III/IV

### 5. 典型例题

**例**：设计 $\omega=\pi$ 处增益非零的高通，选哪类？

Type I（$N$ 奇，偶对称）或 Type II；Type III 在 $\omega=\pi$ 为零。

### 6. 考试要点

- 四类 FIR 对称性与适用滤波器
- 理解 Type III 不能低通/高通的原因
- 由 $h(n)$ 判断类型
- $H(e^{j\omega})$ 线性相位形式

### 8. 幅频与零极点

线性相位 FIR 的 $H(e^{j\omega})=e^{-j\omega(N-1)/2}H_r(\omega)$，$H_r(\omega)$ 全为实。零点成对镜像对称于单位圆（实系数）。设计时检查 Type 约束避免 $\omega=0$ 强制零点。

### 9. 滤波器选型表

低通 Type I；高通 Type I（$N$ 奇）；带通 Type I/III；希尔伯特变换 Type III/IV。
