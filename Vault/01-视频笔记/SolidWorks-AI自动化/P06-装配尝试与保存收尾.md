## 详细笔记

> 以下内容基于机械 AI + SolidWorks 自动化通用实践撰写，对应 UP 分 P 主题。**非逐字转写**。

### 1. P06：装配尝试与保存收尾

系列最后一 P：将 P04/P05 零件**插入装配体**，添加配合关系，尝试完整流程并**保存交付**。

### 2. 装配 API 基本流程

1. NewAssembly 或 OpenAssembly
2. AddComponent4 插入零件（固定或浮动）
3. AddMate2 添加配合（重合、同心、距离）
4. EditAssembly 重建
5. SaveAssembly 保存 .SLDASM
6. 可选：导出 BOM、STEP、图片

### 3. 配合类型选择

| 配合 | 自由度约束 | 典型用途 |
|------|-----------|----------|
| 重合 Coincident | 面/点重合 | 贴面 |
| 同心 Concentric | 圆柱同心 | 轴孔 |
| 距离 Distance | 面间距 | 间隙 |
| 平行/垂直 | 方向 | 方向对齐 |

**完全定位**：6 DOF 约束至 0（除固定零件）。

### 4. 装配顺序

1. 固定基础零件（底座）
2. 插入主要零件，添加关键配合
3. 插入标准件（螺栓等）
4. 检查干涉 InterferenceDetection
5. 爆炸视图（可选，演示用）

### 5. 保存与交付

- 装配体 .SLDASM + 零件文件夹
- 打包 Pack and Go（便携）
- 导出 STEP 用于下游 CAM/仿真
- 截图或 eDrawings 分享

### 6. 全流程回顾（P01–P06）

```
图纸阅读 → 建模思路 → 环境准备 → API 调试 → 零件生成 → 标准件/复杂件 → 装配保存
```

### 7. 局限与扩展

- 本系列为**流程演示**，工业级还需 PLM、版本管理、批量 BOM
- AI 读图错误仍需人工复核
- 可扩展：有限元、工程图自动生成、MCP 驱动 SolidWorks

### 8. 实践要点

- 配合前先 Fix 基准零件
- 避免过约束配合
- 装配失败时逐个 Suppress 配合定位问题
