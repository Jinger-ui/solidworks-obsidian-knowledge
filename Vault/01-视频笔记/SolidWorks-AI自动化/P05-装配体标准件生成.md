## 核心要点

1. P04 的基础零件在此 P **追加复杂特征**（筋、阵列、异形切除等）。
2. **标准件**（螺栓、轴承、键等）通常从 Toolbox/库引入，而非全手工建模。
3. 完善后的零件集合是 P06 装配的输入。
4. AI 需识别图纸中的标准件标注（规格、国标号）并映射到库零件。

## 详细笔记

### 1. 复杂零件完善（预期）

- 在 P04 零件上添加次要特征
- 处理图纸中的细节：圆角、倒角、筋板、薄壁等
- 验证特征顺序对外形的影响（避免后续装配干涉）

### 2. 标准件处理（预期）

| 类型 | 常见处理方式 |
|------|-------------|
| 紧固件 | SolidWorks Toolbox 生成螺栓螺母垫圈 |
| 轴承/密封件 | 设计库或供应商模型 |
| 型材/管材 | 配置驱动或库零件 |

预期流程：

1. 从图纸识别标准件规格（如 M8×25）
2. 通过 API 或界面插入 Toolbox 零件
3. 配置材料、等级、长度等参数
4. 保存为标准件实例文件

### 3. 零件集整理（预期）

- 建立零件清单（BOM 雏形）
- 统一命名：`零件号_名称.sldprt`
- 检查各零件坐标系是否便于装配配合

### 4. 与装配的准备

- 确认每个零件的 **配合面/轴线** 已明确
- 复杂零件简化（如螺纹用装饰螺纹而非真实建模）
