## 详细笔记

> 以下内容基于机械 AI + SolidWorks 自动化通用实践撰写，对应 UP 分 P 主题。**非逐字转写**。

### 1. P04：基础零件建模生成

在 API 流程确认后，按 P01 建模思路**自动生成基础零件**：拉伸、旋转、切除、简单孔。重点是**参数化特征树**与**可重复脚本**。

### 2. 特征建模顺序（最佳实践）

1. **基体特征**：ExtrudeBoss、RevolveBoss
2. **去除材料**：ExtrudeCut、RevolveCut
3. **工程特征**：HoleWizard 或简单切除孔
4. **修饰**：Fillet、Chamfer
5. **阵列**：LinearPattern、CircularPattern

原则：先大后小、先整体后局部、阵列前完成单实例。

### 3. 参数化设计

- 尺寸命名与 JSON 键一致（如 `"D@草图1"` 对应 `flange_od`）
- 方程式驱动关联尺寸（可选）
- 配置（Configuration）管理多规格（若系列零件）

### 4. 草图约束要点

- 优先几何约束（同心、垂直）再标注尺寸
- 对称零件用中心线 + 对称约束
- 避免悬空几何

### 5. 脚本结构建议

```
load_params(json)
connect_sw()
new_part()
sketch_base(params)
extrude_boss(depth)
cut_holes(hole_list)
pattern_holes(count, pcd)
fillet_edges(radius)
save(path)
```

### 6. 质量检查

- 质量属性 MassProp（体积、质量）
- 干涉检查（多零件时）
- 与图纸关键尺寸逐项核对

### 7. 实践要点

- 一个脚本对应一类零件，而非一张图一脚本（可配置化）
- 保留中间 STEP 导出便于非 SW 环境查看
- 失败时回滚到上一成功特征
