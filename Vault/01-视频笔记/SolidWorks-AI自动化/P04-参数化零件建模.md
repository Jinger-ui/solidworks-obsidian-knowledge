## 核心要点

1. 在 API 流程确认后，开始 **自动生成基础零件**——教程的第一个实质性建模产出。
2. 「基础零件」指主体几何已由草图+拉伸/旋转等标准特征构成，尚不含复杂修饰。
3. AI 提供的参数在此 P 真正驱动 SolidWorks 特征树生长。
4. 应关注 **模型保存规范**（文件命名、版本、路径），为 P05/P06 做准备。

## 详细笔记

### 1. 基础零件建模流程（预期）

1. 接收 P03 确认过的参数集
2. 选择零件模板，创建新 `PartDoc`
3. 按建模思路依次创建：
   - 草图（Sketch）
   - 基体特征（Boss-Extrude / Revolve 等）
   - 辅助特征（孔、倒角等基础项）
4. 重建模型（Rebuild）并验证质量属性

### 2. 参数驱动建模（预期）

| 步骤 | API/操作 | 输入 |
|------|----------|------|
| 建草图 | `SketchManager` | 平面、轮廓坐标 |
| 拉伸 | `FeatureExtrusion` | 深度、方向 |
| 修改尺寸 | `Parameter` | 图纸解析值 |

### 3. 生成与保存（预期）

- 实时预览模型是否符合图纸主视图/剖视图的预期
- 保存为 `.sldprt`，建议放入 `Projects/` 对应子目录
- 记录特征树结构，供 P05 追加复杂特征

### 4. 学习检查点

- [ ] 能否仅凭参数 JSON 复现基础零件？
- [ ] 修改一个参数后模型是否自动更新？
- [ ] 保存的零件能否在 SolidWorks 中独立打开？
